# Docker Task Manager

A task management application built using:

- Flask
- MySQL
- Nginx
- Docker Compose
- Bootstrap

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Task Completion
- Priority Levels
- Due Dates
- Overdue Detection
- Dashboard Analytics

## Run

```bash
docker compose up --build

Application:
http://localhost
